7.1 Tasa de registro

Se quiere obtener la tasa de registro (TR) del ganado vacuno y de los perros domésticos para dos tipos de resolución; a nivel de cámara y a nivel de sistema.

La TR para cualquier especie se calcula con la siguiente expresión.

\[\textrm{TR} = \frac{\textrm{Total registros independientes}}{\textrm{Esfuerzo de muestreo}}\]

En el caso del cálculo a nivel de sistema tanto el esfuerzo como el total de registros es la sumatorio de las tres cámaras. Cuando se calcula a nivel de cámara se utilizan el total de registros y esfuerzo para la misma.

Pasemos al calculo de este parámetro, primero se cargan los registros de las camaras trampa.

source("functions/load_data.R")
data <- load_data()

Luego se filtran solo los registros de B. taurus y de C. familiaris, se crea una columna llamada sistema (system) y se seleccionan las columnas de interés.

tr_data <- data %>%
  filter(sp %in% c("Btau", "Cfam")) %>% 
  mutate(system = str_sub(camera, 1, 4)) %>% 
  select(system, camera, sp)
paged_table(head(tr_data))

7.1.1 Nivel sistema

En primer lugar se busca el esfuerzo para cada sistema.

system_effort <- read_rds(file = "data_processed/camera_data.RData") %>% 
  mutate(system = str_sub(camera, 1, 4)) %>%
  select(system, camera, effort) %>% 
  group_by(system) %>%
  drop_na() %>% 
  summarise(effort = sum(effort))

Esfuerzo de todos los sitemas.

Luego se contabiliza la cantidad de registros independientes de cada especie en los sistemas.

tr_system <- tr_data %>%
  select(system, sp) %>%
  group_by(system, sp) %>%
  summarise(count = n()) %>%
  ungroup()

Y se realiza el cálculo.

tr_system <- tr_system %>%
  left_join(system_effort, by = join_by(system)) %>%
  group_by(system, sp) %>% 
  summarise(tr_system = count/effort)

7.1.2 Nivel camara

En primer lugar se carga el esfuerzo para cada camara.

camera_effort <- read_rds(file = "data_processed/camera_data.RData") %>%
  select(camera, effort) %>% 
  drop_na()

Se cuenta la cantidad de regitros independientes en cada cámara para las especies de interés.

tr_camera <- tr_data %>%
  select(camera, sp) %>%
  group_by(camera, sp) %>%
  summarise(count = n())

Se realiza el calculo de la tasa de registro.

tr_camera <- tr_camera %>%
  left_join(camera_effort, by = join_by(camera)) %>%
  group_by(camera, sp) %>% 
  summarise(tr_camera = count/effort) %>% 
  mutate(system = substr(camera, 1, 4))

7.1.3 Tabla única

Se unen los datos en una tabla única para mejor organización, en ella existen los valores de la tasa de registro tanto a nivel de sistema como de camara.

tr_results <- tr_camera %>% 
  left_join(tr_system, join_by(system, sp)) %>% 
  select(system, camera, sp, tr_system, tr_camera)

7.2 Coberturas

En este capítulo se detallará el procesamiento de los datos de coberturas de hábitat, así como la variable de la diversidad de hábitat mediante el índice de Shannon (Hsist).

Las clases de coberturas se pueden dividir en dos tipos:

7.2.1 Clases de Cobertura Simples

Se refieren a categorías específicas de uso del suelo y vegetación.

Clase Descripción
PLNT Plantaciones forestales (incluye mayormente plantaciones sensu stricto, en menor medida cortinas forestales, raramente sombra ornamental y frutales)
FRST Monte Nativo (en muchos casos incluye montes nativos con presencia de exóticas, así como parches de monte de Acacia)
SHRB Matorrales y Pajonales (incluye arbustales, áreas arboladas abiertas de nativas o exóticas con dosel discontinuo, pajonales, caraguatales y juncales altos)
OPEN Áreas Abiertas (praderas, pasturas sembradas, cultivos, dunas, playas, arenales, suelo o roca desnudo, caminos)
VWTR Vegetación inundada o en cuerpos de agua
OWTR Cuerpos de agua abiertos
URBN Áreas urbanas construidas (no se utiliza esta información, en cambio, se prefiere el uso del conteo de techos)

7.2.2 Clases de Cobertura Agregadas

Estas combinan categorías simples para representar de manera más generalizada los diferentes tipos de cobertura de hábitat.

Clase Descripción
WTLD Humedales (VWTR + OWTR)
CLSD Vegetación cerrada (PLNT + FRST + SHRB)
WOOD Áreas Arboladas, de dosel continuo/denso (FRST + PLNT)

Por otro lado, el índice de Shannon (Hsist) se utiliza para medir la diversidad de hábitat. Este índice se estima a partir de las clases de cobertura simple, proporcionando una medida cuantitativa de la heterogeneidad del paisaje. Un valor alto del índice de Shannon indica una mayor diversidad de hábitat.
