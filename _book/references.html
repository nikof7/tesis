11.5 Paquete brms

  • El paquete brms ajusta modelos bayesianos generalizados multivariados y multinivel usando Stan.
  • La sintaxis de fórmulas es similar a lme4, facilitando su uso.
  • Soporta diversos tipos de modelos:
    • Lineales, robustos y de conteo.
    • Supervivencia, tiempos de respuesta y ordinales.
    • Modelos con inflación de ceros, de obstáculo y mezclas definidas por el usuario.
  • Ofrece características adicionales:
    • Términos no lineales y suaves.
    • Estructuras de autocorrelación y datos censurados.
  • Permite predecir todos los parámetros de la distribución de respuesta.
  • El ajuste del modelo se evalúa mediante:
    • Verificaciones predictivas posteriores.
    • Validación cruzada de dejar uno fuera (LOO).
load("data_processed/datos_procesados_v4.RData")
data <- data %>% 
  select(camera, sp, solar, open_cam)

#data$solar <- circular(data$solar, units = "radians", template = "none")
data_filtered <- data %>%
  filter(sp == "Lgym")

percs <- quantile(data_filtered$open_cam , probs = c(0.33, 0.66))

data_filtered$open_cam_lvls <- cut(data_filtered$open_cam, breaks = c(-Inf, percs[1], percs[2], Inf), labels = c("Bajo", "Medio", "Alto"))
tmp <- data_filtered %>%
  filter(open_cam_lvls == "Bajo")

low <- fitact(tmp$solar, sample="model", reps=100)

tmp <- data_filtered %>%
  filter(open_cam_lvls == "Medio")

mid <- fitact(tmp$solar, sample="model", reps=100)

tmp <- data_filtered %>%
  filter(open_cam_lvls == "Alto")

high <- fitact(tmp$solar, sample="model", reps=100) 
plot(low, yunit="density", data="none", las=1, lwd=2,
     cline=list(lty=0)) # Elimina intervalos de confianza

plot(mid, yunit="density", data="none", add=TRUE, 
     tline=list(col="red", lwd=2),
     cline=list(lty=0))
plot(high, yunit="density", data="none", add=TRUE, 
     tline=list(col="red", lwd=2),
     cline=list(lty=0))
